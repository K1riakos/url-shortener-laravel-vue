## 🏠 Run Locally

Clone the project

```bash
  git clone https://github.com/K1riakos/url-shortener-laravel-vue.git
```

Go to the project directory

```bash
  cd url-shortener-laravel-vue
```

Install npm dependencies

```bash
  npm install
```
